# ASBP Design Gate Checklist (Pre-Milestone)

## 1. Scope Alignment

- [ ] Milestone scope clearly defined
- [ ] No hidden scope creep

## 2. Dependency Awareness

- [ ] Cross-WP dependencies considered
- [ ] Impact of dependencies understood

## 3. Scheduling Readiness

- [ ] Scheduling assumptions validated
- [ ] No conflicting time logic

## 4. Calendar Readiness

- [ ] Calendar assumptions explicit
- [ ] No hardcoded single-calendar dependency

## 5. State Logic Stability

- [ ] Task state transitions stable
- [ ] No temporary hacks/workarounds

## 6. Automation Boundaries

- [ ] System vs user responsibilities stable

## 7. Data Structure Safety

- [ ] No hardcoded assumptions about presets/task pools size

## 8. Export Integrity

- [ ] Outputs will reflect system logic correctly

## 9. Known Risks Logged

- [ ] Any incomplete design decisions recorded

## 10. Go / No-Go Decision

- [ ] Safe to proceed without future rework risk
